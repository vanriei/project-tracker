Project Tracker

Two Step Performance

Sign in with your Google account to access your personal project workspace. Your projects are private — no one else on the team can see them.

Only @twostepperformance.com accounts can sign in.

Setup Guide — do this once before sharing
Step 1 — Get a free Google OAuth Client ID (~5 min) Go to console.cloud.google.com
→ Create a new project (or use an existing one)
→ APIs & Services → Credentials → + Create Credentials → OAuth 2.0 Client ID
→ Application type: Web application
→ Under "Authorized JavaScript origins" add your hosted URL (e.g. https://yourname.github.io)
→ Click Create → Copy the Client ID
Step 2 — Add your Client ID to this file Open Project_Tracker.html in a text editor.
Find the line near the top of the <script> that says:
googleClientId: 'YOUR_GOOGLE_CLIENT_ID…'
Replace the placeholder with your actual Client ID.
Step 3 — Host on GitHub Pages (free) 1. Create a free account at github.com
2. Click + New repository → name it anything → set to Public
3. Upload your Project_Tracker.html file (rename it to index.html first)
4. Go to Settings → Pages → Source: Deploy from a branch → main → / (root)
5. Your URL will be: https://yourusername.github.io/repo-name
6. Add this URL to the Authorized JavaScript Origins in Step 1
Step 4 — Share the URL Send your team the GitHub Pages URL. Anyone who signs in with their @twostepperformance.com Google account will get their own private workspace automatically.
